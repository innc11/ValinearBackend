# AwesomeCommentServer

一个有后端静态页面评论系统

**本项目仍处于测试版本，建议随时备份数据以免丢失**

CSS部分参考了[xCss/Valine](https://github.com/xCss/Valine)，后端部署需要PHP环境（>7.2），数据使用Sqlite存储，方便随时迁移，所有数据自己持有，可以任意备份或者折腾。

更多信息请参考本项目对应的前端项目：https://github.com/innc11/VaComment
